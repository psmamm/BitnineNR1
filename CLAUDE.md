## TECHNICAL GUIDELINES

### Code Standards

```typescript
// ALWAYS: TypeScript strict mode
// NEVER: any types (unless absolutely unavoidable)
// ALWAYS: Error handling on every async operation
// NEVER: Console.logs in production code
// ALWAYS: Parameterized queries (prevent SQL injection)
// NEVER: Secrets in code (use Cloudflare Secrets)
```

### Security Requirements

- AES-256-GCM encryption for API keys
- PBKDF2 key derivation (100,000 iterations)
- Firebase Authentication with 2FA
- All secrets in Cloudflare Workers Secrets
- No sensitive data in logs or error messages

### Deployment Commands

```bash
# Development
npm run dev           # Frontend (localhost:5173)
npm run dev:worker    # Backend (localhost:8787)

# Production
npm run build         # Build frontend
wrangler deploy       # Deploy backend
npx wrangler pages deploy dist  # Deploy frontend

# Database
wrangler d1 execute circl-db --file=./migrations/XX_name.sql
```
